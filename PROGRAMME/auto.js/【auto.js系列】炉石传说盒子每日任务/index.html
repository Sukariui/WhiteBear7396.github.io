# 为什么选择 Auto.js

Auto.js 是个基于 JavaScript 语言运行在 Android 平台上的脚本框架。Auto.js 主要工作原理是基于辅助服务 AccessibilityService。Auto.js 是利用安卓系统的 “辅助功能” 实现类似于按键精灵一样，可以通过代码模拟一系列界面动作的辅助工作。因为是开源框架所以安全性很高，他能在手机上模拟人的重复繁琐的工作，不打破被执行的 APP 规则，不修改，不破坏被执行的 APP，可以放心使用。官方的入门文档写得比较全面，可以进行直接上手。
