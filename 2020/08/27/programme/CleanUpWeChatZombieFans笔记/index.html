# 匿名函数

（（）=> {} ）、(function(){})
标准的函数，没有赋值给任何变量，没有名字的函数，叫匿名函数。主要是利用函数内的变量作用。避免产生全局变量，影响整体页面环境，增加代码的兼容性。因为没有名字，所以在定义完成就要调用，（（）=> {} ）（），后面的（）是调用运行这个匿名函数的意思.
